# Automation Flow

## Overview

Script Generation → Content Creation → Posting → Tracking

---

## Steps

1. Generate scripts using AI
2. Insert into Canva template
3. Export video or image
4. Save locally or to cloud folder
5. Post or schedule
6. Track performance
